# Gaussian Elimination Matrix Solver

This project is a C program that uses Gaussian elimination to solve matrix problems.

The main focus of this implementation is applying row operations to transform matrices and compute results such as determinants and inverses.

## Features

- Performs Gaussian elimination (row reduction) 
- Computes determinant using elimination
- Computes matrix inverse using row operations
- Supports matrices up to 10 x 10
- Detects singular (non-invertible) matrices

## How to Run

Compile and run using:

```bash
gcc main.c -o matrixsolver
./matrixsolver
```
## Description
This implementation demonstrates how Gaussian elimination can be used to systematically reduce a matrix and extract useful results.
## Note
This repository focuses specifically on the Gaussian elimination implementation.
